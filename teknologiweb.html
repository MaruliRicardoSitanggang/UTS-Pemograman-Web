Mengenal HTML, CSS, JavaScript & Tailwind CSS

Empat teknologi utama yang digunakan dalam pembuatan website modern. HTML membentuk struktur, CSS mempercantik tampilan, Tailwind mempercepat desain, dan JavaScript membuat website menjadi interaktif.

HTML

HTML (HyperText Markup Language) adalah dasar setiap halaman web. Ia membangun struktur seperti teks, gambar, tabel, dan tautan agar dapat ditampilkan di browser.

<h1>Halo Dunia</h1>
<p>Selamat datang di web saya.</p>

CSS

CSS (Cascading Style Sheets) mempercantik tampilan halaman. Dengan CSS, kamu bisa memberi warna, mengatur ukuran, jarak, dan membuat animasi agar web terlihat menarik.

p {
  color: blue;
  font-size: 18px;
}

Tailwind CSS

Framework CSS modern berbasis utility class yang mempercepat desain tanpa menulis CSS baru. Sangat populer di kalangan developer modern.

<div class="bg-blue-500 text-white p-4 rounded">
  Halo</div>

JavaScript

Bahasa pemrograman yang menambahkan interaktivitas dan logika pada web. JavaScript dapat menampilkan pesan, memvalidasi form, hingga membuat animasi.

document.querySelector ("button").onclick = () => alert("Hai!");
